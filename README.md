# Packers & Movers Landing Page

This repository contains a **responsive landing page** for a Packers & Movers service built using **HTML, CSS, and JavaScript**. The page includes a Hero section, Services section, Contact form, and Footer. It is fully mobile responsive, SEO-friendly, and includes smooth animations using AOS.

## Features

- **Header / Navbar**: Sticky navigation with links to all sections.
- **Hero Section**: Full-width banner with call-to-action button.
- **Services Section**: Three service cards (Home Moving, Office Relocation, Storage & Packing) with hover effects and scroll animations.
- **Contact Form**: Fields for Name, Phone Number (validated for 10 digits), Service Selection, and Submit button.
- **Footer**: Company info and social media links.
- **Animations**: Smooth scroll animations with AOS library.
- **Mobile Responsive**: Optimized for tablets and smartphones.
- **Clean Code Structure**: Separate CSS and JS files for maintainability.

## Installation / Usage

1. Clone or download this repository.
2. Open `index.html` in a browser to view the landing page.
3. Folder Structure:

packers-movers/
│
├── index.html
├── css/
│ └── style.css
├── js/
│ └── script.js
└── images/
└── hero.jpg

## Deployment

- You can host this page on **GitHub Pages** or **Netlify** for a live demo.
- Contact form includes JS validation for mobile numbers, but does not have a backend integration.
